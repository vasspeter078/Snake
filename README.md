# Snake
Snake játék C++ nyelven, objektum-orientált programozási elveket követve.

A fejlesztés alatt tovább ismerkedtem a programozás alapjaival. Megtanultam bánni a objektumokkal, osztályokkal, örökléssel, összetettebb algoritmusokkal, összetettebb dinamikus memóriakezeléssel, generikus szerkezetekkel.
